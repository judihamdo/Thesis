description: Random example that shows how do we construct a new MixedType Instance. When it is from DataType: we can either fill it with assignment through fill command or when we want to construct a new instance of it we use the command new for it. But  when it is from LiteralType or any other primitiveType we use only fill command. The Output here is a random Animal, that will be created in the return_hole

data: Predator(name: str, deceitful: bool)

data: Prey()

type: Pet=Literal['cat', 'dog']

type: Animal = Pet| Predator| Prey| str

signature: main: (Animal, Pet) -> Animal

intro: a

intro: p

let: a1: Animal

fill: a

let: a2: Animal

new: Prey

let: a3: Animal

fill: p

let: a4: Animal

new: Predator

fill: 'Tiger'

fill: True

return:
